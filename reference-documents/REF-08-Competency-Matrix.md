## 1. Purpose

Defines required competencies for each role in the DDD unit and tracks current competency levels to identify training needs, as required by ISO 9001:2015 Clause 7.2.

## 2. Competency Scale

| Level | Description | Indicator |
|-------|-------------|-----------|
| 0 | No knowledge | Has no exposure to the topic |
| 1 | Awareness | Understands concepts, cannot perform independently |
| 2 | Basic | Can perform with guidance/supervision |
| 3 | Proficient | Can perform independently |
| 4 | Expert | Can teach others, drives improvements in the area |
